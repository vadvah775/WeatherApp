# WeatherApp

Веб-приложение для получения погоды с использованием внешнего API (OpenWeather).  
Проект написан на Spring MVC (без Spring Boot), использует Hibernate, PostgreSQL и Tomcat 11.

Подробное описание задания и шаблон веба взяты: https://zhukovsd.github.io/java-backend-learning-course/Projects/TennisScoreboard/

## Технологии

- Java 17
- Spring MVC (XML конфигурация)
- Hibernate ORM
- PostgreSQL
- Maven
- Tomcat 11
- JSP/JSTL для представлений

## Требования для локального запуска

- JDK 17
- Maven 3.6+
- PostgreSQL 16+
- Tomcat 11 (или встроенный tomcat7-maven-plugin)
